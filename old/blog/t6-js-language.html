JavaScript Language

In the JavaScript language, describe all the use cases for each of these:

parentheses ()
  1. Execute a function var result = doSomething()
  2. Hold parameters of a function function addTogether(x, y){ return x+y; }
  3. Used in mathematic equations function doMath(x, y, z){ return (x+y)/z; }
brackets []
  1. Create an array var array = [] or var array = [contents0, contents1, contents2]
  2. Access an entry in an array var firstItem = array[0]
braces {}
  1. Used to denote that multiple lines of code are part of a single packet. This can be used in situations such as if statements or functions (as above) if(somethingIsTrue){ doSomething(); doAnotherThing(); }
  2. Braces are also used to contain the variables in an object constructor var person = {firstName:"John", lastName:"Doe", age:50, eyeColor:"blue"};
double quotes ""
  1. These are most commonly used to denote a String variable var s = "I'm a String!"
single quotes ''
  1. Can be used in similar ways as double quotes var s = 'This is also a String.'
  2. Useful when you want to use double quotes within a string"var codeBit = 'firstName:"John", '
